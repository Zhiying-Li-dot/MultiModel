# Etsy 产品样例数据

从 Etsy 收集的电商产品视频和图片样例，用于定义 PVTT 任务边界。

## 数据列表

| 商品名称 | 类型 | 文件 |
|---------|------|------|
| Family Charm Necklace | 首饰 | `family-charm-necklace-custom-birthstone.*` |
| Personalized Couple Bracelets | 首饰 | `personalized-couple-bracelets-settwo.*` |
| Winter Ski Gear Pillow Cover | 家居 | `winter-ski-gear-pillow-cover-12x20-inch.*` |

## 文件说明

每个商品包含：
- `.webp` - 产品主图 (1588×1588)
- `.mp4` - 产品展示视频

## 数据来源

Etsy 电商平台，2026年1月收集。

## 用途

1. 分析电商产品视频的常见风格和运镜
2. 定义 PVTT 任务的输入输出边界
3. 作为初步实验的测试数据
