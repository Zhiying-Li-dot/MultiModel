## 学习建议

### 多模态大模型
适合有一定深度学习基础的学习者，重点理解：
- Vision Transformer 与 LLM 的结合
- 多模态对齐与融合机制
- 指令微调与 RLHF

### AIGC算法
从生成模型基础开始，逐步深入：
- 先理解 GAN、VAE、Diffusion 三大生成范式
- 重点掌握 Stable Diffusion 原理
- 了解各模态生成的特点与挑战
